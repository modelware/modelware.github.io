#### Description

The goals of this project is to develop a rigorous method and toolset for architecting multi-disciplinary (e.g., GN&C, power) autonomous system behavior that can be efficiently implemented, analyzed, and tested (using simulations). The methdology adopts the State Analysis architectural pattern and formalizes it in [OML](http://www.opencaesar.io/oml/).
